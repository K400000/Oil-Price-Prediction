## üìÅ File Descriptions

### 1. `processed/` (Ready for Modeling)
Contains the **Master Dataset** that has been cleaned, merged, and transformed. **Use this file for training your models.**

* **File:** `merged_oil_prices.csv`
* **Description:** A combination of Brent and WTI prices, aligned by date.
* **Column Dictionary:**

| Column Name | Meaning | Brand / Source | Role in AI Model |
| :--- | :--- | :--- | :--- |
| **`date`** | Trading Date (MM/DD/YYYY) | - | Key for merging |
| **`close_x`** | **Closing Price** | **Brent (Target)** | **Target ($Y$)** |
| `open_x` | Opening Price | Brent | Feature ($X$) |
| `high_x` | Highest Price | Brent | Feature ($X$) |
| `low_x` | Lowest Price | Brent | Feature ($X$) |
| `volume_x` | Trading Volume | Brent | Feature ($X$) |
| **`close_y`** | **Closing Price** | **WTI (Support)** | Feature ($X$) |
| `open_y` | Opening Price | WTI | Feature ($X$) |
| `high_y` | Highest Price | WTI | Feature ($X$) |
| `low_y` | Lowest Price | WTI | Feature ($X$) |
| `volume_y` | Trading Volume | WTI | Feature ($X$) |

> **üí° Note on Suffixes:**
> * **`_x`** = Data from **Brent Crude** (The primary dataset).
> * **`_y`** = Data from **WTI Crude** (The supporting dataset).

---

### 2. `raw/` (Original Source)
Contains the original CSV files downloaded from Kaggle.
‚ö†Ô∏è **Warning:** Never edit files in this folder manually. If the data is messed up, we lose the original reference.

* `brent_prices.csv`: Historical Brent Crude prices.
* `wti_prices.csv`: Historical WTI Crude prices.
