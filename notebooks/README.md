# ЁЯУУ Jupyter Notebooks (Model Creation Zone)

This directory is the **core workspace for creating and training AI models**. 

## ЁЯОп Main Purpose (р╕зр╕▒р╕Хр╕Цр╕╕р╕Ыр╕гр╕░р╕кр╕Зр╕Др╣Мр╕лр╕ер╕▒р╕Б)
These notebooks are specifically designed to:
1.  **Build & Train:** Feed the processed data into algorithms to create the AI models.
2.  **Evaluate:** Check the accuracy (RMSE) of each model.
3.  **Export:** Save the trained models (`.pkl`) to the `models/` directory for future use.

## ЁЯзк Notebooks List

### 1. ЁЯФН Data Analysis
* **`EDA.ipynb` (Start Here!):** **Exploratory Data Analysis.**
    * Used for visualizing trends and checking correlations between Brent and WTI prices *before* we start training.

### 2. ЁЯдЦ Model Creation (Files for building models)
* **`MLR.ipynb`**: **Creates the Linear Regression Model.**
    * Trains the model using Scikit-learn and saves it as `mlr_model.pkl`.
* **`RF.ipynb`**: **Creates the Random Forest Model.**
    * Performs hyperparameter tuning (finding the best settings) and saves the best model as `rf_model.pkl`.
* **`NN.ipynb`**: **Creates the Neural Network Model.**
    * Designs the network layers, trains the model, and saves it as `nn_model.pkl`.

> **Note:** After running these notebooks, you will find the saved model files in the `models/` folder.
